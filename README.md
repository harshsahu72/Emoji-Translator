# Emoji Translator ✨

A premium, fast, and completely offline web application that instantly translates your text into emojis in real-time. Built with a beautiful, dynamic user interface containing a hardcoded dictionary of over 3,000 emojis.

## Features

- ⚡️ **Real-time Translation**: Instantly see emojis appear as you type.
- 🎨 **Premium UI/UX**: Modern glassmorphism design with animated background mesh gradients and polished typography (Inter & Outfit fonts).
- 📶 **Offline Capable**: Completely standalone application with no external API dependencies. All emojis are bundled locally in the JavaScript dictionary.
- 🔄 **Translation Toggle**: Easily toggle the translation feature on or off using the intuitive switch.
- 🚀 **Lightning Fast**: Vanilla JavaScript implementation for maximum performance without the overhead of heavy frameworks.

## Tech Stack

- **HTML5**: Semantic web structure
- **CSS3 (Vanilla)**: Custom styling including mesh gradients, flexbox, and responsive design
- **JavaScript (Vanilla)**: Core logic, DOM manipulation, and real-time translation handling
- **FontAwesome**: High-quality SVG icons
- **Google Fonts**: Modern typography (Inter & Outfit)

## Getting Started

Because this is a vanilla web application, running the project is incredibly simple.

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Start typing in the text area and watch your words transform into emojis!

## Project Structure

```
├── index.html    # Main entry point and page structure
├── style.css     # Premium UI styling and animations
├── script.js     # Translation logic and event listeners
└── emojiDB.js    # Hardcoded dictionary containing thousands of emojis
```

## Usage

Type a sentence into the main text area. Words that match known objects, feelings, or concepts in the dictionary will automatically have their corresponding emoji appended or substituted based on the app's rules.

Example: "I want to eat pizza" -> "I want to eat pizza 🍕"
